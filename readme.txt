== Description ==

Control where to automatically inject your ad rotator shortcode into your WordPress website content.

Posts, pages, or both. Choose how far in, use any shortcode.

== Installation ==

1. Upload `wbp-ad-filter-injector` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Find the controls in the Plugins menu.
4. Save your settings.

== Frequently Asked Questions ==

= Can I use any ad rotator plugin? =

Technically, you can use any shortcode for any plugin. Our design was for injecting an ad rotator shortcode, site-wide and save time.
